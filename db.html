Database Schema POST TABLE ID: blog post id userID: User id to identify who made the post or left the comment timestamp: Timestamp of when post was made title: Title of blog post category: Category of the blog post content: Text of actual post USER TABLE ID: ID to match POST userID of post table displayName: User alias password: User password userEmail: User email COMMENTS TABLE ID: Comment ID userID: Corresponds with USER TABLE ID postID: Corresponds with POST TABLE ID content: Content of comment timestamp: Time comment posted
