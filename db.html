Database Schema POST TABLE ID: blog post id User_ID: User id to identify who made the post or left the comment Timestamp: Timestamp of when post was made Title: Title of blog post Category: Category of the blog post Content: Text of actual post USER TABLE ID: ID to match POST User_ID of post table Display Name: User alias Password: User password EMail: User email COMMENTS TABLE ID: Comment ID User_ID: Corresponds with USER TABLE ID Post_ID: Corresponds with POST TABLE ID Content: Content of comment Timestamp: Time comment posted
